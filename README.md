## Activity Flow

```
┌─────────────────────────────────────────────────────────┐
│ 1. Thêm sản phẩm vào cửa hàng                           │
│    AddProduct() → products: 100 | Warehouse: 0          │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│ 2. Bán hàng (trừ trực tiếp từ products)                 │
│    Bán 52 → products: 48 | Warehouse: 0                 │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│ 3. Hủy cửa hàng (Dispose)                               │
│    Đẩy 48 còn lại về kho → products: 0 | Warehouse: 48  │
└─────────────────────────────────────────────────────────┘
```
